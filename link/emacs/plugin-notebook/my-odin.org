* odin-mode
Major mode for editing source code files written in the Odin programming
language.

#+begin_src emacs-lisp
  (require 'straight)

  (use-package odin-mode
    :mode ("\\.odin\\'" . odin-mode)
    :straight
    (odin-mode :type git :host github :repo "mattt-b/odin-mode"
               :branch "master"))
#+end_src
